## ?? Getting started

Clone the project and access the folder

```bash
$ git clone https://github.com/arthurvenicio/nlw6-letmeask.git && cd nlw-letmeask
```

Follow the steps below

```bash
# Install the dependencies
$ yarn install

# Start the project
$ yarn start
```
